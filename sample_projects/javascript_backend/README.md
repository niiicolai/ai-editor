# To-do App
A simple JavaScript To-do backend application.

# Dependencies
- MongoDB
- Express
- Dotenv
- Node.js
- Vitest

# Install
Configure environment variables
```bash
cp .env.example .env
```

Install dependencies
```bash
npm i
```

# Usage
Start server
```bash
npm start
```

# Test
Run vitest
```
npm test
```
