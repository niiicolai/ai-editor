# User App
A simple TypeScript user application.

# Dependencies
- MongoDB
- Express
- Dotenv
- Node.js
- Vitest
- Bcrypt

# Install
Configure environment variables
```bash
cp .env.example .env
```

Install dependencies
```bash
npm i
```

# Usage
Start server
```bash
npm start
```

# Test
Run vitest
```
npm test
```
